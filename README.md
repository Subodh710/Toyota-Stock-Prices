Toyota Motor Corporation Stock Price Dataset (1980–2026)
📌 Project Overview
This repository contains a comprehensive historical dataset of Toyota Motor Corporation (TM) stock prices, covering over 45 years of trading history. The data includes daily Open, High, Low, Close (OHLC) prices and trading Volume, providing a robust foundation for financial modeling, algorithmic trading development, and machine learning research.

📊 Dataset Specifications
Time Range: March 18, 1980 – January 30, 2026

Frequency: Daily

Total Records: 11,500+ rows

Columns:

Date: The trading date (YYYY-MM-DD).

Open: Price at the start of the trading session.

High: Maximum price during the session.

Low: Minimum price during the session.

Close: Final price at market wrap-up.

Volume: Number of shares traded.
